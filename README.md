## 🔐 ตั้งค่า User / Password

### Default credentials
- Username: **`admin`**
- Password: **`lekden123`**

### วิธีเปลี่ยน password ของตัวเอง

**ขั้นที่ 1:** สร้าง hash ของรหัสผ่านใหม่

1. เปิด https://emn178.github.io/online-tools/sha256.html
2. ในช่อง **Input** พิมพ์ในรูปแบบ: `username:password:salt`
   - ตัวอย่าง: `admin:mysecret123:lottoai_2026_salt`
3. กด **Hash** → copy ตัวอักษรยาวๆ (64 ตัว) ที่ได้

**ขั้นที่ 2:** เอา hash ใส่ใน `index.html`

1. เปิด `index.html` ใน GitHub → กด **Edit** (ปากกา)
2. หา section `LOGIN_CONFIG`
3. ใน `validHashes`: แทน hash เดิมด้วย hash ใหม่ของคุณ
4. กด **Commit changes**

```js
window.LOGIN_CONFIG = {
  enabled: true,
  salt: "lottoai_2026_salt",  // เปลี่ยน salt ก็ได้ (จะปลอดภัยขึ้น)
  validHashes: [
    "ใส่_hash_ใหม่_ของคุณตรงนี้",  // 64 ตัวอักษร
  ],
  title: "เลขเด่น Analytics",
  subtitle: "เข้าสู่ระบบเพื่อใช้งานแดชบอร์ด",
};
```

### เพิ่ม user หลายคน

ใส่หลาย hash ในอาเรย์ `validHashes`:

```js
validHashes: [
  "hash_user_1",  // user1
  "hash_user_2",  // user2
],
```

### ปิดระบบ login (ใครเข้าก็ได้)

```js
enabled: false,
```

## ⚠️ เรื่อง Security ที่ควรรู้

GitHub repo เป็น **public** = code ของเว็บใครก็ดูได้ → คนที่รู้เทคโนโลยี **อาจ crack hash ได้** ถ้าใช้ password ง่ายๆ

**คำแนะนำเพื่อความปลอดภัย:**
- ✅ ใช้ password ยาว 12+ ตัวอักษร + ตัวเลข + สัญลักษณ์
- ✅ เปลี่ยน `salt` เป็นคำของคุณเอง (เช่น ชื่อแมว + วันเกิด)
- ✅ อย่าใช้ password เดียวกับเว็บอื่น
- ❌ อย่า commit password ลงในไฟล์อื่น
- 🔒 ระบบนี้เพียงพอกัน "คนทั่วไปเจอ URL บังเอิญ" ไม่เพียงพอกัน hacker จริงจัง

## 🎯 อธิบายสูตร "เลขวิน 4 ตัว"

ระบบใช้ 3 สูตรหาเลขเด่น แล้วแตกเป็นหลักเดียว:

### สูตร 1: 📅 ตามวัน
- ดูสถิติว่าในวันที่จะออก (จันทร์-อาทิตย์) — เลขไหนออกบ่อยที่สุด
- ใช้ข้อมูล 3 ปีย้อนหลัง

### สูตร 2: 🔥 ออกบ่อย
- ดูสถิติทั้ง period ว่าเลขไหนออกบ่อยที่สุด
- ไม่สนใจวัน

### สูตร 3: 🌊 เลขไหล
- ดู 20 งวดล่าสุด — เลขไหนปรากฏบ่อย (มีน้ำหนัก)
- งวดใหม่กว่ามีน้ำหนักมากกว่างวดเก่า

### การคำนวณเลขวิน 4 ตัว
1. ได้เลขเด่น 3 ตัวจาก 3 สูตร (เช่น `12`, `45`, `25`)
2. แตกเป็นหลักเดียว: `1`, `2`, `4`, `5`, `2`, `5` (รวม 6 หลัก)
3. นับความถี่: `2` ออก 2 ครั้ง, `5` ออก 2 ครั้ง, `1`, `4` ออก 1 ครั้ง
4. เรียงตามความถี่: `2`, `5`, `1`, `4`
5. **เลขวิน 4 ตัว: 1, 2, 4, 5** (เรียงน้อย→มากเพื่ออ่านง่าย)

## 🎯 ฟีเจอร์เลือกงวด — Re-check ความแม่นยำ

ที่ส่วน "งวดที่จะวิเคราะห์" สามารถ:
- คลิก **งวดถัดไป** = ทำนายงวดที่ยังไม่ออก
- เลือกงวดในอดีต = ทดสอบสูตรกับผลที่รู้แล้ว

**เมื่อเลือกงวดในอดีต:**
- ระบบจะใช้ข้อมูลแค่ก่อนงวดนั้น (ไม่ cheat)
- เปรียบเทียบกับผลจริง บอกว่าตรง X หลักจาก Y หลัก
- หลักที่ตรงจะมีกรอบสีเขียวพร้อม ✓

วิธีนี้ช่วยประเมินว่าสูตรนี้แม่นแค่ไหน

## 🐛 Troubleshooting

**🔴 Login ไม่ผ่าน แม้ใส่ admin/lekden123 ถูก**
- ตรวจสอบว่าไม่ได้แก้ `salt` ในไฟล์
- ถ้าแก้ salt → ต้อง hash ใหม่ทั้งหมด

**🔴 ลืม password**
- ตั้งใหม่ตามขั้นตอน "วิธีเปลี่ยน password"

**🔴 หวยใหม่ดึงไม่ได้**
- รัน workflow ใหม่ที่หน้า Actions
- ถ้า 0 งวด → URL exphuay.com อาจเปลี่ยน บอกผมจะแก้

**🔴 อยากให้ไม่ต้อง login**
- ตั้ง `enabled: false`
